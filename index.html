What you’ll learn

A clear, structured path into cloud engineering with AWS, Python and AI — suitable for students, instructors and working professionals.

Foundation
AWS core & security

Understand IAM, users, roles and policies the way real teams use them. Learn how to keep accounts safe, separate environments, and avoid common mistakes in student or production setups.

Automation
Python + boto3 in practice

Use Python to automate EC2, S3, tagging, monitoring and more. Replace manual clicking with scripts that are testable, repeatable and ready for DevOps and CI/CD pipelines.

Serverless

Monitoring
Cost & operations

Create simple cost-monitoring and alerting tools using CloudWatch metrics and billing data so learners understand the real operational side of running systems in AWS.
