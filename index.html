Core focus areas

The AliTechGrid catalog centres on three competency tracks that can be used as standalone offerings or combined into a broader cloud-adoption program.

Track 1 · Cloud Foundations & Governance

AWS accounts, IAM & secure baselines

Establish the guardrails: accounts, IAM, MFA, least privilege, tagging and basic cost awareness. Ideal for teams standardising their AWS usage or formalising “lab” environments.

  • Account structure, identity and access patterns.
  • Safe lab and sandbox configuration.
  • Practical governance through tagging and policies.
Track 2 · Python for Cloud Automation

From console clicks to code

Use Python and the AWS SDK to automate inventory, tagging, reporting and operations workflows. Replace manual work with repeatable, reviewable automation.

  • Python essentials for cloud engineers.
  • boto3 patterns for inventory and tagging.
  • Simple reporting and operational scripts.
Track 3 · Serverless & AI on AWS

Modern workloads, practical scope

Build small but meaningful solutions with Lambda, API Gateway, managed data services and AI/ML APIs, focused on what practitioners can actually deploy.

  • Serverless APIs and event-driven patterns.
  • Data-backed services and monitoring.
  • AI workloads integrated into applications.
