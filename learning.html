Modules & learning outcomes

Each module can be a week in a course, a unit in your LMS or a self-study milestone. Link your own slides, worksheets and quizzes to these modules.

Module 1
Cloud & IAM Foundations
Goal: understand accounts, regions and identity.
  • AWS global infrastructure: regions, AZs, services.
  • Accounts, root user and lab-safe best practices.
  • IAM users, groups, roles and policies.
  • Hands-on: create a dedicated lab IAM user with MFA.
  • Outcome: student can safely log in and follow any lab.
Module 2
Python + boto3 Automation
Goal: replace console clicks with code.
  • Configuring credentials & profiles for boto3.
  • Listing EC2 instances and S3 buckets from Python.
  • Filtering by tag, region and state.
  • Hands-on: build an inventory script writing CSV.
  • Outcome: student can read AWS state programmatically.
Module 3
Serverless APIs with Lambda
Goal: deploy logic without managing servers.
  • Lambda basics: runtime, handler, memory, timeout.
  • Event-driven architecture intro.
  • API Gateway HTTP APIs to expose Lambda.
  • Hands-on: build a JSON “hello” API & test via browser/Postman.
  • Outcome: student can publish a simple serverless API.
Module 4
DynamoDB & Observability
Goal: store data and observe systems.
  • DynamoDB tables, keys, read/write capacity.
  • CRUD from Python / Lambda using boto3.
  • CloudWatch logs & metrics for Lambda and APIs.
  • Hands-on: log requests into DynamoDB and build an error alarm.
  • Outcome: student can persist data and troubleshoot failures.
Module 5
AI Workloads & Mini-Project
Goal: connect AI services to real cloud flows.
  • Overview: Comprehend, Rekognition, Transcribe, Bedrock.
  • Hands-on: sentiment analysis or document classification job.
  • Optional: simple chatbot or analysis API using Lambda.
  • Mini-project: combine IAM, Lambda, API Gateway, DynamoDB & AI.
  • Outcome: student has a portfolio-ready demo.

10-lab pack (student & instructor versions)

These labs are designed to align with your videos and worksheets. Later you can link PDF / DOCX / notebook versions directly from each item.

  1. Lab 1 – Account hygiene & IAM user setup Create a lab-only IAM user, attach least-privilege policy and enable MFA.
  2. Lab 2 – First boto3 script Install boto3, configure credentials and list EC2 instances & S3 buckets.
  3. Lab 3 – Tagging automation Write a script to apply Owner / Environment tags across EC2 and S3.
  4. Lab 4 – Deploying your first Lambda function Create, test and monitor a basic Python Lambda that returns JSON.
  5. Lab 5 – Lambda + API Gateway HTTP API Expose Lambda via HTTPS, secure it and call it from browser/Postman.
  6. Lab 6 – DynamoDB CRUD Create a table, insert items and query them from Lambda.
  7. Lab 7 – Logging & alarms with CloudWatch Break something on purpose and use logs/alarms to trace it.
  8. Lab 8 – Cost & usage snapshot Use AWS tools or APIs to understand where cost would appear.
  9. Lab 9 – Text analysis with AI Send text to an AI service and interpret the output as part of a workflow.
  10. Lab 10 – Mini-project integration Combine IAM, Lambda, API Gateway, DynamoDB and AI into an end-to-end demo.
