How these labs are designed

Each lab is scoped so it can be completed within a focused session, with clear prerequisites, student deliverables, and instructor notes.

Student experience

  • Step-by-step tasks and screenshots in PDF or DOCX format.
  • All labs reference the same AWS account and IAM setup.
  • Students finish with a concrete artifact (script, API, dashboard or report).
  • Difficulty ramps from basic console tasks to automation and AI.

Instructor & LMS use

  • Each lab can map to a single grade item in Brightspace / Moodle.
  • Instructor version includes answer keys, rubrics and troubleshooting notes.
  • Labs can be combined into mini-projects or semester-long capstones.
  • Easy to localise: only AWS console region and screenshots need updating.
